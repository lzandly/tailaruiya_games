# 卡通泰拉瑞亚 Web 基础版

这是根据项目 PDF 产品文档转译实现的本地可玩原型。当前仓库同时保留 Web 原型和 Godot 4.x 工程，后续可继续扩展到完整的 NPC、Boss、更多生态和道具系统。

## Godot 工程

- 入口文件：`project.godot`
- 主场景：`scenes/Main.tscn`
- 核心脚本：`scripts/Main.gd`
- 转换说明：`GODOT_PORT.md`

安装 Godot 4.2+ 后，打开 `project.godot` 运行主场景即可。

## 运行方式

### Web 原型

直接用浏览器打开 `index.html` 即可运行。

如浏览器限制本地文件访问，可在项目目录启动静态服务器：

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:8000/`。

## 当前版本功能

- 程序生成横版开放世界，包含地表、泥土层、岩石层、洞穴、矿石和树木。
- 泰拉瑞亚式森林地表卡通渲染，包含昼夜天空、远山、云层、森林剪影、地表树林、蘑菇、萤火点光和雨天效果。
- 玩家移动、跳跃、重力、地形碰撞和受伤/死亡重生。
- 鼠标左键使用当前道具：镐挖掘、斧砍树、剑近战、弓发射木箭；右键放置当前方块，支持距离限制。
- 真实物品栏：铜镐、铜短剑、铜斧、木弓、木箭、火把、工作台、矿石、凝胶和基础方块。
- 物品说明：鼠标悬浮到物品格后显示名称、类型、伤害、工具力和使用说明。
- 背包面板：`E` 打开，展示工具、材料、弹药和可放置物。
- 背包整理：物品栏前 10 格和背包 40 格共享槽位，可拖拽交换，也可打开背包后点击两个格子交换。
- 合成面板：`C` 打开，支持工作台、火把、木弓、木箭、生命上限护符配方。
- 基础敌人刷新、追踪、跳跃、近战伤害和击退。
- 游戏化 HUD，包含心/星状态栏、快捷栏、小地图和操作提示。
- 泰拉瑞亚式界面布局：物品栏固定在左上角，生命/魔力固定在右上角，小地图位于右上状态栏下方。
- 菜单栏：右下角菜单按钮收纳游戏指导，避免常驻遮挡画面。
- 蓝发冒险者角色动作：行走摆臂、跳跃倾斜、挥剑攻击、挥镐挖掘，并随快捷栏切换手中道具。

## 操作

- `WASD` 或方向键：移动
- `Space`：跳跃
- 鼠标左键：使用当前道具
- 鼠标右键：放置当前方块
- `1-0` 或滚轮：切换快捷栏
- `E`：打开/关闭背包
- `C`：打开/关闭合成
- 背包打开时点击两个物品格：交换位置
